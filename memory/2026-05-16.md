## 部署 — Railway → Google Cloud Run

### 決策
Railway 免費方案用量有限制。Google Cloud Run free tier：2M req/月，scale-to-zero，不超限不收費。

### 設定
- GCP Project：`hearth-491015`（既有帳號，已有帳單帳戶）
- Region：`asia-east1`
- Service URL：`https://financial-pdf-reader-647193990749.asia-east1.run.app`
- UI：`https://financial-pdf-reader-647193990749.asia-east1.run.app/ui`
- Memory：512Mi

### 新建檔案
- `.dockerignore`：排除 .env、memory/、docs/、__pycache__ 等
- Dockerfile 小修：`COPY requirements.txt .`、`COPY . .`（相對路徑）

### MongoDB Atlas 注意事項
- Network Access 必須設 `0.0.0.0/0`（Cloud Run IP 動態）
- 密碼避免特殊字元（PowerShell 環境 `$` 等會被截斷）
- 部署後更新 env vars：`gcloud run services update financial-pdf-reader --region asia-east1 --set-env-vars "KEY=VAL,..."`

### 部署指令（code-only redeploy）
```powershell
cd "d:\financial-pdf-reader"
gcloud run deploy financial-pdf-reader --source . --region asia-east1
```
