## Phase 2A — UI Semantic Color System

### 核心原則
`review-required ≠ risk`，不確定性不等於危險

### 修改
- `b-review`：藍色（#0369a1 / #f0f9ff）— 待確認，非風險
- `attn-tag-risk`：紅色 — 真正結構風險
- `attn-tag-watch`：橘色 — 待觀察條件
- `attn-tag-info`：藍色 — 不確定性待確認
- `attn-tag-oc`：紫色 — OC 規則完整性警告
- Attention Panel 各 item 使用語意 tag，而非舊的單一 `attn-tag`

## 設計哲學（用戶反饋重點）

- 目標：epistemic UI（認知管理），而非 sentiment UI（情緒化顯示）
- Cognitive Hierarchy：先結論 → 原因 → 細節
- Primary Narrative 是 interpretation，需 Narrative Explainability Layer（待做）
- Review type taxonomy 是 epistemic failure mode 分類，不是重要性排序
- 下一步：Narrative Explainability Layer + Review Severity Model (L1-L4)
